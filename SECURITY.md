# Security Policy

## Supported Versions

Actualmente solo se brinda soporte a la versión más reciente del sistema.

| Version | Supported |
|----------|-----------|
| Latest   | ✅ |
| Older    | ❌ |

---

## Reporting a Vulnerability

Si se identifica una vulnerabilidad de seguridad en cualquiera de los sistemas desarrollados por UNIT, esta deberá ser reportada de forma privada.

No publiques vulnerabilidades mediante Issues o Pull Requests.

Contacto:

Email: security@unit.pe

La vulnerabilidad será analizada y corregida lo antes posible.

Gracias por contribuir a mantener la seguridad del proyecto.
